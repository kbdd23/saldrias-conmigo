## ¿Saldrías Conmigo?

Experiencia interactiva para pedirle a alguien especial salir contigo construida con HTML/CSS y JS vanilla.

---

## Caracteristicas

-Botón Si que se mueve de manera inteligente

-Botón "No" que huye y esconde del usuario de manera creativa

-Botón de accesibilidad claro/oscuro

-Selección de actividades y horarios

-Comprobante final
